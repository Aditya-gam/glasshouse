# Flow — Account & Data Rights

> **Depends on:** `06-api/endpoints/account.md`, `08-security-privacy/dsar-export-erasure.md`, `ethics-and-tone.md` · **Version:** v1

Control + transparency.

- **Consents** — view/grant/revoke (purpose, Art. 9, decoy).
- **Retention** — toggle retain-encrypted ↔ **process-then-discard**.
- **Connected accounts** — view/revoke (clears the encrypted token).
- **Export** (DSAR) — download your data bundle.
- **Delete account** — **crypto-shred + cascade**, with the **irreversibility** explained plainly (and the honest caveat that copies others made can't be recalled).
- **Crisis/safety resources** — surfaced for at-risk users / high-severity findings.
